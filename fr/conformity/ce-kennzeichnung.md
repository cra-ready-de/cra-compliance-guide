# Marquage CE selon le CRA : conditions et procedure

## Que signifie le marquage CE selon le CRA ?
    
        - CE = Conformité Européenne : le fabricant declare la conformite avec toutes les regles UE applicables.
        - Sous le CRA, le marquage CE montre que le produit respecte les exigences de securite de l Annexe I.
        - Base juridique : Art.28-31 CRA.
        - Applicable a partir du : 11 December 2027 pour tous les nouveaux produits.
        - **IMPORTANT :** Le marquage CE est obligatoire. Aucun produit sans évaluation de la conformité finalisee ne peut porter le CE.
    

    ## Conditions prealables a l apposition du marquage CE
    
        - Realiser une analyse des risques de cybersécurité (Art.13 Abs.2 CRA).
        - 
            Classer le produit (Default/Classe I/II/Critique) :
            [ /help/cra-grundlagen/produktklassifizierung](/help/cra-grundlagen/produktklassifizierung)
        
        - 
            Choisir la procedure d évaluation de la conformité applicable :
            [ /help/konformitaet/konformitaetsbewertung](/help/konformitaet/konformitaetsbewertung)
        
        - Etablir la documentation technique (Annexe VII CRA).
        - 
            Etablir une SBOM (Annexe I Partie II) :
            [ /help/sbom/sbom-anforderungen](/help/sbom/sbom-anforderungen)
        
        - Etablir la déclaration de conformité UE (DoC) (Art.28 CRA).
        - Apposer le marquage CE sur le produit ou dans la documentation d accompagnement (Art.30 CRA).
    

    ## La declaration de conformite UE (DoC)
    Contenu obligatoire selon Art.28 Abs.2 CRA :

    
        - Nom et adresse du fabricant
        - Identification du produit (nom commercial, modele, type, numero de serie)
        - Declaration de la responsabilite exclusive du fabricant
        - Objet de la declaration (produits comportant des éléments numériques)
        - Reference aux normes harmonisees appliquees ou aux specifications communes
        - Le cas echeant : nom et numero d identification du Notified Body
        - Lieu, date, signature
        - Conservation : au moins 10 ans (Art.28 Abs.3 CRA)
    

    ## Apposition du marquage CE (Art.30 CRA)
    
        - Sur le produit lui-meme, ou sur l emballage, ou dans les documents d accompagnement (si l apposition physique est impossible).
        - Taille minimale : 5 mm de hauteur (sauf pour les tres petits produits).
        - Aucun autre marquage pouvant etre confondu avec CE.
        - Avec un Notified Body : ajouter le numero d identification a quatre chiffres apres le marquage CE.
    

    ## Surveillance du marche et consequences
    
        - Les autorites nationales de surveillance du marche (en Allemagne : BSI) controlent la conformite CE (Art.35 CRA).
        - Faux marquage CE : amende jusqu a 15 millions EUR ou 2,5% du chiffre d affaires annuel (Art.64 Abs.1 CRA).
        - Un rappel de produit est possible en cas de defauts graves.
        - **Remarque :** La pratique d execution varie entre les Etats membres.
